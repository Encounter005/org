#+title:  奈奎斯和香农定理

* 失真

在传输过程中与原有信号有所偏差

** 影响失真程度的因素
+ 码元传输速率
+ 信号传输距离
+ 噪声干扰
+ 传输媒体质量

* 码间串扰

信道带宽是信道能通过的最高频率和最低频率之差


* 奈氏准则

1. 理想情况下极限数据传输率为$2Wlog{2}V(b/s)$
   + W是信道带宽，单位时Hz
   + 极限码元传输速度位是2WBaud
   + V是离散电平数目
2. 在任何信道中，码元传输的速率是有上限的
3. 给出了码元传输速率的限制，但并没有对信息传输速率限制
4. 要提高数据的传输速率，使用多元制的调制方法（让每个码元携带更多的信息）


* 香农定理

信噪比（db） = $10log{10}(S/N)$

信道的极限数据传输速率为$Wlog{2}(1 + S/N)(b/s)$

+ 信道的带宽或信道中的信噪比越大，则信息的极限传输速率就越高
+ 对一定的传输带宽和一定的信噪比，信息传输速率的上限就确定了
+ 只要信息的传输速率低于信道的极限奈传输速率，就一定能找到某种方法实现无差错传输
+ 实际信道的传输速率要比香农定理算出来的低不少
+ 若信道带宽W或信噪比没有上限（不可能），那么信道的极限信息传输速率也就没有上限


* 结论

1. 给定了信道的带宽，则该信道的极限波特率就确定了，不可能超过这个极限波特率传输码元，除非改信道的带宽
2. 增加信道比特传输速率
   + 可以增加改信道的带宽
   + 选择更高的编码方式
